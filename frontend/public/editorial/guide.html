Onboarding & Deployment

Webサイト公開・操作ガイド

Open Justice Editorialへようこそ。このガイドでは、作成したコンテンツをWebサイトとして世界に公開するための5つのステップを解説します。

1

コードを確認する

編集画面の右上にあるツールバーから「</> View Code」ボタンを探してください。ここから生成されたHTMLソースコードにアクセスできます。

code View Code

※ 編集が完了したら必ずこのボタンをクリックして最新のコードを生成してください。

ads_click
2

ソースをコピーする

表示されたモーダルウィンドウ内のコードを全選択してコピーします。「Copy to Clipboard」ボタンを使用すると正確かつ迅速にコピーが可能です。

3

ローカルに保存

お使いのテキストエディタ（VS Code等）を開き、新規ファイルを作成して貼り付けます。ファイル名は必ず index.html として保存してください。

description
index.html
4

多言語フォルダ構成

日本語版と英語版を両方公開する場合は、ディレクトリを分けます。ルートディレクトリに /jp と /en フォルダを作成し、それぞれの index.html を配置します。

folder_open project_root
folder jp / index.html
folder en / index.html
5

サーバーへアップロード

最後にファイルをGitHubリポジトリにプッシュします。VercelやAWS Amplifyと連携していれば、プッシュと同時に自動で世界中に公開されます。

cloud_done GITHUB
arrow_forward
rocket_launch VERCEL / AWS
